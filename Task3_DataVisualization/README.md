# Task 3 - Data Visualization

## Objective

Transform raw book dataset into meaningful visualizations to discover insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizations Created

* Rating Distribution Bar Chart
* Price Distribution Histogram
* Average Price by Rating
* Rating Percentage Pie Chart
* Price Spread Boxplot

## Key Insights

* Rating patterns were identified.
* Book price distribution was analyzed.
* Relationship between ratings and prices was explored.
* Data storytelling was improved through visual charts.
